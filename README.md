# hermes-van

Cliente web para [Hermes Agent](https://github.com/NousResearch/Hermes-Agent). Reemplazo de Telegram para uso diario.

Landing con explicación y roadmap: [`index.html`](./index.html) — publicado en GitHub Pages.

## Qué es

Webapp que se conecta directo al API server nativo del gateway de Hermes (puerto 8765) y expone:

- Multi-chat real con streams SSE independientes
- Slash commands con palette sourced en vivo
- Auth WebAuthn + recovery codes
- Compatibilidad total: cero modificaciones al backend

## Estado

Phase 0 — Foundation. Ver el roadmap en `index.html`.
